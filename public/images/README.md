## Image Guidelines

- **Format:** JPG or PNG
- **Size:** 1200x1200px or larger recommended
- **Aspect Ratio:** Square (1:1) or Portrait (3:4) works best
- **File Size:** Compress images to reduce load time (aim for < 500KB each)

## Tips

- Use high-quality photos for best results
- Ensure good lighting in photos
- Consider using WebP format for better compression (Next.js will optimize automatically)
- All images will be automatically optimized by Next.js Image component
