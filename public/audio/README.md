# Audio Directory

Place your background music file here.

## Required File

**wedding-music.mp3** - Background music for the wedding invitation

## Supported Formats

- MP3 (recommended)
- WAV
- OGG

## Audio Guidelines

- **Duration:** Any length (will loop automatically)
- **File Size:** Compress to reasonable size (aim for < 5MB)
- **Format:** MP3 at 128-192 kbps is recommended for web
- **Content:** Instrumental or soft background music works best

## Tips

- Use royalty-free music or music you have rights to use
- Test the audio sync with animations
- Consider providing multiple quality options for different connection speeds
- The audio will loop automatically when it ends

## Audio Synchronization

The audio timeline is used to sync animations throughout the website. Key sync points:
- Hero section animations start at ~0.5s
- Love story timeline animations trigger at intervals
- Photo gallery animations are time-based

You can adjust these timings in the respective component files if needed.
